# 🚀 快速开始指南

## 5 分钟上手指南

### 第一步：安装依赖

```bash
# 1. 安装 Python 依赖
pip install -r requirements.txt

# 2. 确保已安装 Graphviz（用于拓扑图布局）
# Windows: choco install graphviz
# macOS: brew install graphviz
# Linux: sudo apt-get install graphviz
```

### 第二步：启动应用

```bash
streamlit run app.py
```

浏览器会自动打开到 `http://localhost:8501`

### 第三步：导入示例数据

1. 在侧边栏点击"上传 CSV 文件"
2. 选择项目中的 `example_tasks.csv` 文件
3. 查看自动解析的任务列表

### 第四步：分析结果

点击 **"🔍 开始分析与优化"** 按钮，查看：

- 📊 **关键指标**：总工期、关键路径任务数
- 📈 **甘特图**：红色=关键路径，蓝色=非关键路径
- 🕸️ **拓扑图**：任务依赖关系可视化

## 💡 核心概念速查

### 关键路径（红色）
- ✅ 决定项目最短工期
- ✅ 必须优先完成的任务链
- ✅ 延迟会直接影响总工期
- ❌ 无法延迟，没有浮动时间

### 非关键路径（蓝色）
- ✅ 有浮动时间
- ✅ 可以适当延迟而不影响总工期
- ✅ 资源紧张时可调整优先级

## 🎯 典型使用场景

### 场景 1：项目管理
识别哪些任务必须按时完成，哪些可以灵活调度。

### 场景 2：流程优化
找到瓶颈任务，通过拆分或增加资源来缩短总工期。

### 场景 3：依赖分析
可视化任务依赖关系，发现潜在的并行机会。

### 场景 4：资源分配
将有限资源优先投入到关键路径任务。

## 📋 CSV 格式速查

```csv
id,duration,deps
任务 A,5,
任务 B,3,A
任务 C,4,"A, B"
```

**要点：**
- `id`：唯一标识符
- `duration`：耗时（秒）
- `deps`：前置依赖（逗号分隔，可空）

## 🔧 常见问题

### Q: 提示"循环依赖"错误
**A**: 检查是否有环形依赖，如 A→B→C→A

### Q: 甘特图不显示
**A**: 确保至少添加了一个有效任务

### Q: 拓扑图报错
**A**: 确认已安装 Graphviz 并添加到 PATH

### Q: 中文显示乱码
**A**: 确保 CSV 文件使用 UTF-8 编码保存
